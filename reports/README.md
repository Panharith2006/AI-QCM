# Reports & Documentation

All reports are auto-generated by the `scripts/generate_reports.py` script.

## How to Generate Reports

```bash
# Generate reports from a test image
python scripts/generate_reports.py --image path/to/test_sheet.jpg --yolo artifacts/yolo/best.pt

# Custom output directory
python scripts/generate_reports.py --image path/to/sheet.jpg --output my_reports
```

## What Each Report Contains

### Layer Reports
- Technical explanation of the algorithm
- Before/after comparison images
- Mathematical formulas and parameters
- Performance statistics and timing
- Why this approach was chosen

### Technology Stack
- Libraries and dependencies
- Architecture decisions (rule-based vs. learning-based)
- Alternatives considered
- Performance characteristics

### Explainable AI (XAI)
- Grad-CAM visualization techniques
- Model interpretation and transparency
- Which regions the model focuses on for each prediction class
- Confidence assessment and anomaly detection
- Validation of model decision-making process

### End-to-End Pipeline
- Complete processing flow diagram
- Overall performance metrics
- Accuracy breakdown by question type
- Known failure modes and mitigations
- Extension ideas for future work

## Customization

Edit template files in each layer directory to add:
- Your own observations
- Test results with different parameters
- Performance analysis
- Comparison with alternative approaches
- Screenshots and annotations
